# Content Pivot Tools

Two lightweight, no-build writing tools (plain HTML/CSS/JS) that turn a topic into a finished, human-sounding piece in four steps: outline, your personal angle, draft, then a cleanup pass that flags AI-sounding phrases and em dashes before an AI polish rewrites just the flagged spots. Drafts autosave locally as "case files" so you can leave one mid-draft and resume later.

## editorial-pivot.html
Built for Cyber Brief. Outlines follow a security-editorial structure (Hook → Technical Proof → your Editorial Pivot → Mechanics → Defensive Playbook) and drafts write for a CISO/security-executive audience with acronyms spelled out and pull quotes marked.

## local-content-pivot.html
Built for contractor and realtor content. Pick a client type (contractor / realtor) and content type (blog post, project/listing spotlight, or social captions); outlines follow a Hook → Credibility Proof → your Personal Touch → Practical Breakdown → Call to Action structure in a warmer, plain-spoken voice.
